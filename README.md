# AI Fake News Detector

An AI-powered Fake News Detection System built using:

- React.js
- FastAPI
- EasyOCR
- DistilBERT
- SQLite

## Features

- Upload news screenshots
- OCR text extraction
- Fake/Real News Detection using DistilBERT
- Confidence Score
- Prediction History
- Delete Prediction Records

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI
- SQLite

AI/ML:
- EasyOCR
- DistilBERT
- PyTorch

## Future Improvements
